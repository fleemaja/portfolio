Poll Vault

Poll Vault is a "content and comment" web app that is a hub for user-submitted polls. Anyone can vote on polls and authenticated users can add polls and comments. The visual design is influenced by Google Fonts, Twitter's polls, and Youtube's comment section. It is a React/Redux web app that consumes a Node/Express RESTful API with careful consideration given to accessibility and Progressive Web App features.

Car Pool

Car Pool is a desktop (requires keyboard inputs) web game made with JavaScript. It uses p5.js for graphics and animation and uses matter.js to handle the 2d physics. The idea is based on Rocket League, a popular video game where cars play soccer against each other, but is based around the game of pool (aka billiards) instead.

U.S. State Animals Map

U.S. State Animals Map is an interactive d3.js map displaying the United States' designated state animals. Hovering over a state reveals a tooltip with that state's designated state animal of a selected category.
